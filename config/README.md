📅 Agenda Tu Turno – Sistema de Gestión de Turnos Automático

Descripción:
Sistema integral de agendamiento de turnos diseñado para negocios de servicios (salud, estética, etc.) que necesitan automatizar reservas, pagos y notificaciones. La aplicación está construida con Node.js y se integra con Google Calendar, enviando notificaciones instantáneas por correo electrónico, garantizando eficiencia y reduciendo errores administrativos.

🚀 Funcionalidades Destacadas
Reservas Inteligentes: Evita solapamientos, valida disponibilidad en tiempo real.
Interfaz Premium: Diseño limpio, responsive y moderno, fácil de usar tanto para clientes como para administradores.
Integración con Google Calendar: Los turnos se registran automáticamente en el calendario del proveedor.
Notificaciones Automáticas: Emails de confirmación para clientes y administradores mediante Nodemailer.
Gestión de Pagos: Reserva con pago integrado, reduciendo ausencias y pérdidas de dinero.
Gestión de Profesionales y Horarios: Permite agregar o modificar horarios de cada profesional de manera sencilla.
Variables de Entorno: Seguridad y privacidad de credenciales mediante archivos .env.
🛠 Tecnologías Utilizadas
Backend: Node.js, Express
Frontend: HTML5, CSS3 (Diseño Custom), JavaScript Vanilla
APIs: Google Calendar API (OAuth2)
Email: Nodemailer
Gestión de Configuración: Dotenv
Control de Versiones: Git / GitHub

🧩 Uso y Flujo
El cliente accede a la aplicación y selecciona día y hora disponible.
La reserva se valida automáticamente y el turno se marca como ocupado (evitando conflictos).
Se envía un correo al cliente y al administrador confirmando la cita.
El cliente puede reprogramar o cancelar hasta 24 horas antes, con devolución automática si corresponde.
Los administradores pueden agregar profesionales, editar horarios y visualizar el flujo completo de reservas.
🌟 Valor Agregado
Automatización completa: Minimiza la intervención manual y errores humanos.
Escalable: Permite agregar nuevos profesionales o comercios fácilmente.
Orientado al usuario: Experiencia fluida y sencilla para todos los involucrados.
Documentado: Fácil de mantener y modificar para cualquier desarrollador futuro.
